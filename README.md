# GNSS Error Forecasting - Kalman Filter Pipeline

Complete state-space modeling and Kalman filtering pipeline for GNSS satellite error prediction.

## Quick Start

```bash
# Install dependencies
pip install pandas numpy matplotlib

# Run full pipeline
python main.py --mode all --data-folder data

# Run specific modes
python main.py --mode tune       # Train models only
python main.py --mode forecast   # Generate predictions only
python main.py --mode evaluate   # Evaluate predictions only
```

## Pipeline Overview

**8D State-Space Model:**
- State: [X, Ẋ, Y, Ẏ, Z, Ż, Clock, Clock_drift]
- Constant velocity dynamics
- Direct observation of [X, Y, Z, Clock]

**Features:**
- ✅ Automatic preprocessing (15-min resampling, interpolation)
- ✅ Per-satellite Kalman filter training
- ✅ Automated Q/R tuning via grid search
- ✅ Day 8 forecasting (96 predictions per satellite)
- ✅ Comprehensive evaluation (MAE, RMSE, R², 3D error)
- ✅ Automated visualization generation

## Methodology

1. **Preprocessing**: 15-minute resampling, gap interpolation (max 45 min)
2. **Training**: Days 1-7, grid search over Q scale factors
3. **Forecasting**: Pure prediction mode for Day 8
4. **Evaluation**: MAE, RMSE, R², 3D Euclidean error
